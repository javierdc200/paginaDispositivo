Inicio

He escogido este dispositivo principamente por el gran avance que supone, ya que habilita totalmente a personas tetraplégicas, o con mobilidad casi nula a usar un ordenador y aprobecharlo casi sin dificultad.

Este dispositivo consta de dos partes: la primera una pantalla donde el usuario tendra la interfaz de la aplicacion que este usando, las dos aplicaciones basiscas de este dispositivo son teclado virtual para escribir o una compuesta por indicadores de dirección para poder mover el cursos y navegar por la pantalla del ordenador.

La segunda parte del dispositivo es una camara especial que se situa debajo de la pantalla y que permite seguir el movimiento ocular del usuario. Esta camara tiene una frecuencia de 60 imagenes por segundo para garantizar la maxima precision.

Para "pulsar" una tecla el usuario debe mirarla durante un tiempo determinado, por defecto de es 1/2 segundo pero este puede ser ajustado por el usuario.

La conexion con el ordenador es fácil y sencilla ya que este dispositivo se conecta con nuestro de pc por un usb que se conecta de manera inalambriaca al dispositivo.

Caracteristicas Principales

Facil calibración

La calibración toma solo 15 segundos

Generación de voz

Genera voz escribiendo un mensage o mediante frases precofiguradas

Conectividad

Integracion facil con un dispositivo movil, permite llamar y enviar mensajes

Versatilidad

Este dispositivo permite controlar cualquier aplicacion con facilidad

Prediccion de palabras

Este sistema aprende del usuario, haciendo conjeturas basadas en el contexto y en el historial del usuario.

Control domestico

Tambien permite la gestion de dispositivos domoticos desde su propia interfaz.
