Inicio

He escogido este dispositivo principalmente por el gran avance que supone, ya que habilita totalmente a personas tetrapléjicas, o con movilidad casi nula a usar un ordenador y aprovecharlo casi sin dificultad.

Este dispositivo consta de dos partes: la primera una pantalla donde el usuario tendrá la interfaz de la aplicación que esté usando, las dos aplicaciones básicas de este dispositivo son teclado virtual para escribir o una compuesta por indicadores de dirección para poder mover el cursor y navegar por la pantalla del ordenador.

La segunda parte del dispositivo es una cámara especial que se sitúa debajo de la pantalla y que permite seguir él movimiento ocular del usuario. Esta cámara tiene una frecuencia de 60 imágenes por segundo para garantizar la máxima precisión.

Para "pulsar" una tecla el usuario debe mirarla durante un tiempo determinado, por defecto es de 1/2 segundo pero este puede ser ajustado por el usuario.

La conexión con el ordenador es fácil y sencilla, ya que este dispositivo se conecta con nuestro PC por un usb que se conecta de manera inalámbrica al dispositivo.

Caracteristicas Principales

Fácil calibración

La calibración toma solo 15 segundos

Generación de voz

Genera voz escribiendo un mensaje o mediante frases precofiguradas

Conectividad

Integración fácil con un dispositivo móvil, permite llamar y enviar mensajes

Versatilidad

Este dispositivo permite controlar cualquier aplicacion con facilidad

Predicción de palabras

Este sistema aprende del usuario, haciendo conjeturas basadas en el contexto y en el historial del usuario.

Control domestico

Tambien permite la gestion de dispositivos domóticos desde su propia interfaz.
